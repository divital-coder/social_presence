## Medical Imaging Research

### Proposed Projects (Northeastern University)

#### A) Foundational Models for Medical Image Segmentation
- Implement models similar to MONAI network architectures
- Test on Medical Decathlon datasets
- Build modular encoders/decoders with Lux.jl and MedPipe3D.jl
- Report dice scores and training/inference time

#### B) Explainability Algorithms for 3D Medical Imaging
- Adapt explainability algorithms for high-memory medical images
- Implement guided backpropagation, occlusion, training data attribution
- Use classic UNet with Lux.jl and MedPipe3D.jl
- Test on Medical Decathlon and PICAI datasets

#### C) Radiomic Feature Extraction
- Implement GLCM, GLSZM, GLRM, NGTDM, GLDM features
- Test on AutoPET dataset (PET/CT)
- Use Turing.jl for probabilistic analysis
- Compare results with PyRadiomics

#### D) Capsule Network for 3D Medical Imaging
- Implement capsule network architecture
- Compare generalizability to classical UNet
- Test on Medical Decathlon dataset
